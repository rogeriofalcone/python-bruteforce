Brute Force SSH em Python
=========================

Projeto de estudo para criação de um software para quebra de senha, utilizando a técnica de força bruta.
Sistema desenvolvido utilizando Python.

Instalando:
git clone https://github.com/paulopinda/Bruteforce.git

cd Bruteforce
./bsh.py --scan <host> <porta>

Pré-requisitos:
Para executar a aplicação deve instalar a biblioteca Paramiko em Python.

Para instalar no Ubuntu:
apt-get install python-paramiko
